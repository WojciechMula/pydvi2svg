========================================================================
                      pydvi2svg & SVGfrags
========================================================================

``pydvi2svg`` converts DVI files (output from TeX) to SVG vector graphics

``SVGfrags`` is utility build on top of ``pydvi2svg``, that 
includes pieces of TeX/LaTeX expressions into existing SVG file.
